**Tools/Languages**

- HTML | CSS | JS

**Highlights**

Responsive flexbox image grid. 

/* HOME  */ (source: index.html)
```
.wrapper {
    display: flex;
    width: 100vw;
    align-content: flex-start;
    margin: 0;
  }

.wrapper img {
    max-width:100%;
    height: auto;
}

.main_image {
    width: 100%;
    height: auto;
    transition: all 0.8s ease;
}

.main_image:hover {
    transform: scale(1.2)
}
```
